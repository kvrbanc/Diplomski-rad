Machine learning architecture

To summaraize your text, the service utilizes a modern machine learning architecture. The architecture uses a deep learining model called BERT, created by researchers at Google in 2018. The structure of the model and the way it is trained gives it a good "understanding of language". Therefore, it can be used to represent words and sentences of a text in a meaningful way. These representations can be used to determine the most descriptive sentences of a text, thus forming its summary. For this purpose, the clustering tehnique called k-means is used.

To read more about the architecture, see Derek Miller's paper.

Customizable model and simple API

The machine learning architecture is implemented in a way that allows customization. It is easy to specify different versions of BERT model the architecture uses. Also, it is possible to specify parts of the model ( called layers ) from which the sentence representations are taken. Since the architecture is composed of different parts, the fuctionality of each part is easily accessible. Therefore, the architecture ( or its parts ) can be used for various purposes. The source code of the architecture can be found in the GitHub repository.

The summarization service provides a simple API. Currently, only one endpoint is supported. It is used to request text summarization. To read more about the API, see the documantation page.
